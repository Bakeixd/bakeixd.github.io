Intro

순수미술 전공으로 시작해 실무 디자인 경험을 통해 UI/UX와 웹디자인 영역으로 확장해왔습니다. 다양한 환경 속에서 문제를 해결하며 사용자 중심의 구조 설계 역량을 키워왔습니다.

단순히 시각적인 결과를 만드는 것을 넘어, 사용자가 서비스를 어떻게 이해하고 이용하는지를 고민하며 더 자연스럽고 직관적인 흐름을 설계하는 데 집중하고 있습니다.

Experience

실무에서 시작된 디자인

대학 시절 근로장학생으로 디자인 업무를 맡으며 실무를 처음 경험했습니다. 단독으로 다양한 작업을 수행하며 툴을 빠르게 익히고, 제한된 환경 속에서도 결과를 만들어내는 문제 해결 역량을 키웠습니다.

기획부터 구현까지

모바일 기반 서비스를 운영하는 환경에서 UI/UX, 콘텐츠, 운영을 함께 경험하며 서비스 전체 흐름을 이해했습니다. 사용자 반응을 기반으로 개선을 반복하며 화면 단위가 아닌 경험 중심의 설계 역량을 확장했습니다.

웹과 퍼블리싱 확장

뉴스레터 제작을 계기로 HTML을 접하고, 디자인을 직접 구현하며 웹 구조에 대한 이해를 넓혔습니다. 협업을 통해 구현 방식을 익히고 디자인과 개발을 연결하는 경험을 쌓았습니다.

실제 서비스 설계 경험

웹 기반 플랫폼 구축 환경에서 메인 화면 디자인을 담당하며 사용자 흐름을 중심으로 구조를 개선했습니다. 정보 전달과 사용 편의성을 함께 고려한 설계를 통해 실무에 적용 가능한 UX 역량을 강화했습니다.

Strength

UX 설계 중심 사고

사용자의 흐름과 문제를 분석하고 구조적으로 해결합니다

실무형 협업 능력

디자인부터 운영, 개발 협업까지 고려한 결과물을 만듭니다

빠른 학습과 실행력
